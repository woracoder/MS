Instructions to run the test cases:

Goto the pintos/src/threads folder
make clean
make (Ignore the errors)

Goto the pintos/src/examples folder
make clean
make

Goto the pintos/src/userprog folder
make clean
make (Ignore the errors)

Goto the pintos/src/filesys folder
make clean
make (Ignore the errors)

Goto the pintos/src/vm folder
make clean
make (No errors here)

Goto the pintos/src/vm/build folder

Run the below command to run the test cases
make check SIMULATOR=--bochs

The test case results will be given out.
The results of the test cases run by me are attached in the file 'testRunResults.txt' in the pintos directory.
